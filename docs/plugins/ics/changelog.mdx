# Changelog — Plugin Calendrier ICS

Ce fichier liste les évolutions, corrections et améliorations du plugin **Calendrier ICS**, par ordre chronologique inverse.

---

## 2026-05-24

### Nouveautés
- **Migration FullCalendar v4 → v6** : remplacement complet de la bibliothèque par les bundles modulaires FC6 (`core`, `daygrid`, `timegrid`, `list`, `interaction`, `locales-all`). Suppression des dépendances moment.js et jquery-ui.
- Nouveau **widget Mini-Cal** (`eqLogic_min`) : grille mensuelle entièrement custom (sans FC), avec indicateurs visuels d'événements par jour (point vert/rouge, compteur), navigation mois précédent/suivant, et tuile flottante au clic sur un jour affichant la liste des événements (heure ou "Toute la journée", titre).
- Option **"Mini-Cal"** ajoutée dans les formulaires de configuration de tous les types de calendrier (url, local, caldav, widget) : permet de choisir entre le widget liste FC6 et le widget grille mini.

### Améliorations
- Le widget `eventCal` injecte désormais correctement les données du prochain événement (`timestamp`, `title`, `allday`, `location`) depuis les commandes Jeedom via `toHtml()`.
- Le thème Jeedom Light/Dark est désormais appliqué également au widget Mini-Cal via les variables CSS `--jee-*`.
- La vue mobile bénéficie des mêmes nouveaux widgets (Mini-Cal et eventCal).

---

## 2026-04-28

### Nouveautés
- Nouveau **widget carte événement** (`eventCal`) pour les équipements de type événement : affichage de la date, heure, titre et lieu du prochain événement dans un style carte avec entête colorée et système d'urgence visuel (couleur selon le délai restant).
- Support complet des **thèmes Jeedom Light et Dark** pour le widget calendrier FullCalendar (`jeedom-theme-utils.css`).

### Améliorations
- Le widget événement gère automatiquement l'état "Aucun événement" (entête grise, message centré).
- Adaptation des couleurs de l'entête selon l'imminence de l'événement : vert (&gt;24h), orange (&lt;24h), rouge (&lt;1h), rouge foncé (&lt;5min), gris (terminé ou aucun).
- Vue mobile disponible pour le widget carte événement.
