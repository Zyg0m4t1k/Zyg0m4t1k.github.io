# Plugin TaskLauncher

Le plugin **TaskLauncher** permet de créer des équipements temporels dans Jeedom pour déclencher automatiquement des **actions**, des **commandes** ou des **scénarios**.

Il permet notamment de gérer :

- ⏰ des **réveils**
- 🕒 des **horloges**
- 📅 des **événements programmés**
- ⏳ des **décomptes à partir d’une date**

---

## ✨ Vue d’ensemble

TaskLauncher peut être utilisé aussi bien pour de simples rappels que pour des automatisations plus avancées.

Exemples d’usage :

- lancer un scénario du matin à heure fixe
- afficher une horloge sur un design ou un dashboard
- programmer une action à une date précise
- déclencher une commande avec délai ou exécution ponctuelle

---

## ⚙️ Installation

L’installation se fait comme pour n’importe quel plugin Jeedom.

### 1️⃣ Installation depuis le Market

1. Installez le plugin depuis le Market Jeedom.
2. Activez le plugin.
3. Ouvrez la page du plugin dans :

```text
Plugins → Organisation → TaskLauncher et activer le plugin
```

### 2️⃣ Accès au plugin

Une fois le plugin activé, vous pouvez créer un nouvel équipement puis choisir son type.

![type](/img/plugins/TaskLauncher/type.png)

## 🚀 Création d’un équipement

Lors de l’ajout d’un nouvel équipement, vous choisissez le type souhaité : réveil, horloge ou événement.

### Types disponibles

- ⏰ Réveil
- 🕒 Horloge
- 📅 Événement

Une fois créé, l’équipement apparaît dans la liste et peut être configuré selon son usage.

---

## 🛠 Configuration des réveils

### Configuration générale

La configuration d’un réveil est simple :

1. définir l’heure d’exécution
2. choisir les jours de répétition
3. sélectionner les actions ou scénarios à lancer
4. enregistrer puis activer le réveil si nécessaire

![Configuration réveil](/img/plugins/TaskLauncher/reveilconf.png)


### Actions différées

Il est également possible de définir des **actions différées**.

![Configuration actions](/img/plugins/TaskLauncher/actiondiff.png)

---

## 🎛 Commandes disponibles sur le widget

Depuis le widget, plusieurs actions sont directement accessibles.

:::tip Actions possibles
Vous pouvez notamment :

- activer ou désactiver la tâche
- modifier l’heure d’exécution
- modifier les actions
- lancer immédiatement les actions
- définir un lancement différé (enmn)
:::

Sur le **widget alternatif**, il est aussi possible de modifier :

- les heures
- les minutes
- les jours
- l’état d’activation du réveil

:::danger Important
Il ne faut pas activer un équipement entre l’heure différée et l’heure effective, sous peine de déclencher immédiatement les actions prévues.
:::

## 🔗 Utilisation dans les scénarios

TaskLauncher peut être utilisé dans d’autres scénarios Jeedom.

Il est possible par exemple de :

- activer un réveil
- désactiver un réveil
- modifier son heure d’exécution

### Format de l’heure

Le format attendu est le suivant :

```text
0720  →  07h20
1830  →  18h30
```

### Fonctions Jeedom

Vous pouvez aussi utiliser des fonctions Jeedom dans vos scénarios.

```text
#time# + rand(0,100)
```

Dans cet exemple, la tâche sera exécutée à l’heure courante du scénario, plus un délai aléatoire entre 0 et 100.

## 🎨 Thèmes Light et Dark

Tous les widgets TaskLauncher s’adaptent automatiquement au thème Jeedom actif (Light ou Dark) via le fichier `jeedom-theme-utils.css`.

Aucune configuration nécessaire : les couleurs, fonds, bordures et ombres s’ajustent automatiquement selon le thème défini dans **Administration → Interface → Thème**.
