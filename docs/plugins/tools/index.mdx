# 🛠️ Plugin Tools

Le plugin **Tools** regroupe plusieurs outils de création d'interface pour Jeedom.

La partie principale documentée ici est **MyHome**, un constructeur d'interfaces basé sur les plans Jeedom. MyHome permet de créer une interface moderne avec des pages, un menu, des thèmes, une palette d'ajout rapide, des cards d'équipements et des widgets spécialisés.

## 🔄 Flux général recommandé

```text
1. Créer ou préparer les widgets/cards nécessaires
2. Créer le projet MyHome
3. Choisir le thème
4. Configurer le menu
5. Créer les pages
6. Utiliser la palette pour ajouter rapidement les éléments
7. Ouvrir les sous-plans Jeedom
8. Positionner, redimensionner et finaliser le design
```
