# 🧠 Bonnes pratiques MyHome

## 📋 Ordre conseillé

```text
1. Créer le projet
2. Choisir le thème
3. Régler le canvas
4. Configurer le menu
5. Créer les pages
6. Ajouter les zones principales
7. Ajouter les widgets globaux
8. Ajouter les équipements/cards
9. Finaliser dans les sous-plans
```

## 🏗️ Commencer par les zones

Pour une interface lisible, il est conseillé de commencer par les zones :

- bandeau haut ;
- colonne latérale ;
- bloc météo ;
- bloc résumé ;
- bloc énergie ;
- bloc sécurité.

Ensuite seulement, ajouter les widgets et cards.

## 📄 Garder une logique par page

Une page doit correspondre à un usage clair.

Exemples :

| Bon découpage | Moins lisible |
|---|---|
| Page Salon | Une seule page avec toute la maison |
| Page Énergie | Mélanger énergie, sécurité et multimédia |
| Page Sécurité | Ajouter toutes les commandes sans structure |

## 🎨 Utiliser le même thème partout

Le thème assure la cohérence.

Il est préférable d'utiliser le même thème pour tout le projet, puis d'ajuster quelques accents si nécessaire.

## ⚠️ Ne pas surcharger l'accueil

La page d'accueil doit donner une vue rapide :

- météo ;
- état sécurité ;
- alertes ;
- résumé pièces ;
- raccourcis principaux.

Les détails doivent être placés dans des pages dédiées.

## 📱 Prévoir tablette et mobile

Le mode responsive permet d'adapter le rendu, mais il faut penser dès le départ :

- largeur du canvas ;
- taille des cards ;
- lisibilité des boutons ;
- espacement entre les éléments ;
- menu utilisable au doigt.
