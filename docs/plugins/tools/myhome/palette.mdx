# 🎯 Palette MyHome

La palette est l'un des éléments les plus importants de MyHome.

Elle permet d'ajouter rapidement des éléments dans une page sans passer par les menus Jeedom classiques.

## 🧩 Principe

```text
Choisir une page cible
        ↓
Choisir un élément dans la palette
        ↓
Ajouter au sous-plan
        ↓
Ouvrir le design Jeedom
        ↓
Positionner et ajuster
```

## 🎯 Page cible

La palette demande d'abord une **page cible**.

Cette page détermine dans quel sous-plan l'élément sera ajouté.

:::warning
Si la mauvaise page est sélectionnée, l'élément sera ajouté au mauvais sous-plan.
:::

## 📦 Types d'éléments disponibles

La palette peut afficher plusieurs familles :

| Famille | Rôle |
|---|---|
| Templates | Éléments basés sur des commandes Jeedom |
| Widgets | Widgets Tools compatibles MyHome |
| Menus | Éléments de navigation |
| Cards | Cards d'équipements |
| Zones | Zones décoratives ou de structure |

## ⚡ Ajout rapide d'équipements

La palette accélère fortement l'ajout d'équipements.

Elle est particulièrement utile pour :

- lumières ;
- prises ;
- volets ;
- thermostats ;
- alarmes ;
- multimédia ;
- capteurs ;
- scénarios.

## 💡 Pourquoi la palette est importante

Sans palette, il faut créer les éléments manuellement dans les plans Jeedom.

Avec la palette :

- l'utilisateur sélectionne une page ;
- choisit un élément ;
- l'ajoute directement ;
- puis ajuste sa position dans le sous-plan.

## ✅ Bon usage

```text
1. Créer les pages
2. Sélectionner la page cible
3. Ajouter les zones de structure
4. Ajouter les widgets globaux
5. Ajouter les cards d'équipements
6. Ajuster dans le sous-plan
```
