# 📄 Pages et sous-plans

Une page MyHome représente un écran de l'interface.

Chaque page est reliée à un **sous-plan Jeedom**.

## 📄 Rôle des pages

Les pages permettent de séparer les usages.

Exemples :

- Accueil ;
- Salon ;
- Cuisine ;
- Chambres ;
- Énergie ;
- Sécurité ;
- Multimédia ;
- Extérieur.

## 🖼️ Rôle des sous-plans

Le sous-plan est l'endroit où sont réellement placés les éléments visuels.

Dans un sous-plan, on positionne :

- équipements ;
- cards ;
- widgets ;
- zones ;
- textes ;
- images ;
- éléments HTML.

:::important
La personnalisation du design se fait dans les sous-plans Jeedom : position, taille, largeur, hauteur, superposition, z-index et finition visuelle.
:::

## ➕ Création d'une page

Depuis l'onglet **Pages** :

1. ajouter une page ;
2. donner un nom ;
3. créer ou associer le sous-plan ;
4. ouvrir le design ;
5. placer les éléments.

## 🎯 Page cible de la palette

Avant d'ajouter un élément depuis la palette, il faut choisir la page cible.

Cela permet à MyHome de savoir dans quel sous-plan ajouter l'élément.

## 🏗️ Organisation recommandée

| Page | Contenu conseillé |
|---|---|
| Accueil | Résumé global, météo, sécurité, scènes principales |
| Salon | Lumières, multimédia, volets, température |
| Cuisine | Lumières, prises, énergie, capteurs |
| Chambres | Volets, chauffage, lumières |
| Énergie | Consommation, production, graphes, états |
| Sécurité | Alarme, ouvrants, caméras, présence |
