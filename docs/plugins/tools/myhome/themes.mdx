# 🎨 Thèmes MyHome

Le thème définit l'apparence générale du projet.

Il influence :

- le fond ;
- les couleurs ;
- les cards ;
- les widgets ;
- les menus ;
- les effets visuels ;
- l'ambiance globale.

## 🎯 Rôle du thème

Un thème MyHome sert à unifier l'interface.

Lorsqu'un thème est appliqué :

- les widgets héritent des variables visuelles ;
- les cards utilisent les couleurs cohérentes ;
- le menu reprend l'accent du thème ;
- les fonds et bordures gardent une cohérence globale.

## 🖼️ Fond du thème ou image personnalisée

Dans le projet, le fond peut être :

| Mode | Usage |
|---|---|
| Fond du thème | Recommandé pour garder une cohérence automatique |
| Image / URL | Utile pour une interface plus personnalisée |

## ⚙️ Édition avancée des variables

L'onglet **Thème** permet de charger, modifier puis appliquer des variables CSS du thème actif.

Actions disponibles :

| Action | Rôle |
|---|---|
| Charger les valeurs | Lire les valeurs du thème actif |
| Appliquer au fichier CSS | Sauvegarder les modifications dans le fichier CSS |
| Réinitialiser | Revenir aux valeurs précédentes |

:::warning
Cette partie modifie l'apparence du thème. À utiliser après avoir validé le rendu général du projet.
:::
