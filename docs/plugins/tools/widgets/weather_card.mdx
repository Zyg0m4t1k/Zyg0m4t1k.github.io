# weather_card

## 📝 Description

Widget météo complet sous forme de card : température, humidité, vent, pression, pluie, lever/coucher du soleil et prévisions J+1 à J+3.

## Compatibilité

| Élément | Valeur |
|---|---|
| Compatible MyHome | Oui |
| Type widget | `weather_card` |
| Largeur par défaut | `320px` |
| Hauteur par défaut | `360px` |
| Dashboard | Oui |
| Mobile | Oui |

## ⚙️ Paramètres

| Paramètre | Défaut | Options | Description |
| --- | --- | --- | --- |
| accent |  |  | Couleur accent de la card météo (vide = accent du thème MyHome) |
| title |  |  | Titre affiché sur la card (vide = nom de l'équipement météo, none pour masquer) |
| show_sun | 1 | 1, 0 | Afficher lever/coucher du soleil : 1|0 |
| show_forecast | 1 | 1, 0 | Afficher les prévisions J+1/2/3 : 1|0 |


## 🏠 Utilisation dans MyHome

1. Ouvrir le projet MyHome.
2. Aller dans l'onglet **Palette**.
3. Choisir la page cible.
4. Ajouter le widget **weather_card**.
5. Ouvrir le sous-plan Jeedom.
6. Positionner et redimensionner le widget.
