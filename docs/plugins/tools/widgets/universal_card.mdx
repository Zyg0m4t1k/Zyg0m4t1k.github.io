# universal_card

## 📝 Description

Widget universel switch / slider

## Compatibilité

| Élément | Valeur |
|---|---|
| Compatible MyHome | Oui |
| Type widget | `universal_card` |
| Largeur par défaut | `360px` |
| Hauteur par défaut | `70px` |
| Dashboard | Oui |
| Mobile | Oui |

## ⚙️ Paramètres

| Paramètre | Défaut | Options | Description |
| --- | --- | --- | --- |
| mode | switch |  | Mode : switch|slider |
| size | normal |  | Taille : normal|large|compact |
| title |  |  | Titre affiché (vide = nom équipement, none pour masquer) |
| icon | fas fa-plug |  | Icône principale (none pour masquer) |
| accent | #62d5ff |  | Couleur accent |
| format | badge |  | Format infos : badge|horizontal |
| icon_on |  |  | Icône bouton ON/Ouvrir (none pour masquer) |
| icon_off |  |  | Icône bouton OFF/Fermer (none pour masquer) |
| icon_stop |  |  | Icône bouton STOP (none pour masquer) |
| label_on |  |  | Label bouton ON/Ouvrir (none pour masquer) |
| label_off |  |  | Label bouton OFF/Fermer (none pour masquer) |
| label_stop |  |  | Label bouton STOP (none pour masquer) |


## 🏠 Utilisation dans MyHome

1. Ouvrir le projet MyHome.
2. Aller dans l'onglet **Palette**.
3. Choisir la page cible.
4. Ajouter le widget **universal_card**.
5. Ouvrir le sous-plan Jeedom.
6. Positionner et redimensionner le widget.
