# color_zone

## 📝 Description

Zone overlay fond plein — couleur accent, opacité et border-radius configurables. Widget décoratif sans commande.

## Compatibilité

| Élément | Valeur |
|---|---|
| Compatible MyHome | Oui |
| Type widget | `color_zone` |
| Largeur par défaut | `300px` |
| Hauteur par défaut | `200px` |
| Dashboard | Oui |
| Mobile | Oui |

## ⚙️ Paramètres

| Paramètre | Défaut | Options | Description |
| --- | --- | --- | --- |
| color | var(--mh-accent) |  | Couleur de fond : var(--mh-accent)|#b878ff|rgba(255,255,255,.3) |
| opacity | 0.15 |  | Opacité de 0 à 1 (ex: 0.10|0.15|0.20|0.30) |
| radius | var(--mh-radius) |  | Border-radius : var(--mh-radius)|0px|8px|18px|50% |
| blur | 0px |  | Flou backdrop : 0px|4px|8px|12px |
| border_color | var(--mh-accent) |  | Couleur du bord : var(--mh-accent)|#62d5ff|#ffffff |
| border_opacity | 0 |  | Opacité du bord de 0 à 1 : 0|0.3|0.5|0.8|1 |
| shadow_opacity | 0 |  | Opacité de l'ombre de 0 à 1 : 0|0.2|0.4|0.6 |


## 🏠 Utilisation dans MyHome

1. Ouvrir le projet MyHome.
2. Aller dans l'onglet **Palette**.
3. Choisir la page cible.
4. Ajouter le widget **color_zone**.
5. Ouvrir le sous-plan Jeedom.
6. Positionner et redimensionner le widget.
