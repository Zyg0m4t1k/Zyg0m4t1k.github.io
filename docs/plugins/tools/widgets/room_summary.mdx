# room_summary

## 📝 Description

Résumé informatif par pièce : température, lumières, prises, volets ou autres résumés Jeedom sélectionnés

## Compatibilité

| Élément | Valeur |
|---|---|
| Compatible MyHome | Oui |
| Type widget | `room_summary` |
| Largeur par défaut | `360px` |
| Hauteur par défaut | `auto` |
| Dashboard | Oui |
| Mobile | Oui |

## ⚙️ Paramètres

| Paramètre | Défaut | Options | Description |
| --- | --- | --- | --- |
| object_ids |  |  | Ids des pièces/objets Jeedom séparés par virgule. Vide = tous les objets visibles |
| types | temperature,light,outlet,shutter |  | Types affichés séparés par virgule : temperature,light,outlet,shutter,opening,door,humidity,power,security... |
| title | Pièces |  | Titre affiché dans la card, none pour masquer |
| icon | fas fa-layer-group |  | Icône du titre, none pour masquer |
| accent | var(--mh-accent) |  | Couleur accent (#rrggbb ou variable CSS) |
| show_status | 1 |  | 1 = affiche le statut Actif/Veille/Alerte |
| transparent | 0 | 0, 1 | 1 = fond transparent, sans bordure ni ombre |
| show_empty | 0 | 0, 1 | 1 = afficher les pièces sans résumé disponible |


## 🏠 Utilisation dans MyHome

1. Ouvrir le projet MyHome.
2. Aller dans l'onglet **Palette**.
3. Choisir la page cible.
4. Ajouter le widget **room_summary**.
5. Ouvrir le sous-plan Jeedom.
6. Positionner et redimensionner le widget.
