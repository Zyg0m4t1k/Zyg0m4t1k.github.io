# overview_card

## 📝 Description

Card résumé d'un objet Jeedom : badges ON/OFF, température, humidité…

## Compatibilité

| Élément | Valeur |
|---|---|
| Compatible MyHome | Oui |
| Type widget | `overview_card` |
| Largeur par défaut | `280px` |
| Hauteur par défaut | `160px` |
| Dashboard | Oui |
| Mobile | Oui |

## ⚙️ Paramètres

| Paramètre | Défaut | Options | Description |
| --- | --- | --- | --- |
| overview_id |  |  | Id de l'objet Jeedom à afficher |
| accent | #62d5ff |  | Couleur accent de la card (#rrggbb) |
| title |  |  | Titre affiché (vide = nom objet, none pour masquer) |
| icon |  |  | Classe FontAwesome (ex: fas fa-bed). Vide = fas fa-home. none = masqué. |


## 🏠 Utilisation dans MyHome

1. Ouvrir le projet MyHome.
2. Aller dans l'onglet **Palette**.
3. Choisir la page cible.
4. Ajouter le widget **overview_card**.
5. Ouvrir le sous-plan Jeedom.
6. Positionner et redimensionner le widget.
