## 🚀 Utilisation générale

Les widgets MyHome sont ajoutés depuis la **palette** dans une page cible.

Flux recommandé :

```text
Palette
  ↓
Choix de la page cible
  ↓
Ajout du widget
  ↓
Ouverture du sous-plan
  ↓
Positionnement et personnalisation
```

:::info
Les dimensions indiquées sont les dimensions par défaut déclarées dans le plugin. Elles peuvent ensuite être ajustées dans le sous-plan Jeedom.
:::
