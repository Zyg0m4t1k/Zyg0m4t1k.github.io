# myhome_zone

## 📝 Description

Zone décorative overlay (contour rectangulaire ou ligne)

## Compatibilité

| Élément | Valeur |
|---|---|
| Compatible MyHome | Oui |
| Type widget | `myhome_zone` |
| Largeur par défaut | `300px` |
| Hauteur par défaut | `200px` |
| Dashboard | Oui |
| Mobile | Oui |

## ⚙️ Paramètres

| Paramètre | Défaut | Options | Description |
| --- | --- | --- | --- |
| mode | overlay |  | Mode : overlay (contour rect) ou linear (ligne) |
| orientation | horizontal |  | Orientation ligne (mode linear) : horizontal|vertical |
| label |  |  | Titre affiché (vide = nom équipement, none pour masquer) |
| color | rgba(255,255,255,0.22) |  | Couleur bordure/ligne et titre |
| border_style | dashed |  | Style : solid|dashed |
| dash_size | medium |  | Tirets : small|medium|large ou valeur pixels ex: 8,4 |
| title_align | left |  | Position du titre : left|center|right |


## 🏠 Utilisation dans MyHome

1. Ouvrir le projet MyHome.
2. Aller dans l'onglet **Palette**.
3. Choisir la page cible.
4. Ajouter le widget **myhome_zone**.
5. Ouvrir le sous-plan Jeedom.
6. Positionner et redimensionner le widget.
