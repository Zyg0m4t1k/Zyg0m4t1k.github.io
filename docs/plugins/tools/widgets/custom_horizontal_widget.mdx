# custom_horizontal_widget

## 📝 Description

Widget horizontal personnalisable

## Compatibilité

| Élément | Valeur |
|---|---|
| Compatible MyHome | Oui |
| Type widget | `custom_horizontal_widget` |
| Largeur par défaut | `360px` |
| Hauteur par défaut | `70px` |
| Dashboard | Oui |
| Mobile | Oui |

## ⚙️ Paramètres

| Paramètre | Défaut | Options | Description |
| --- | --- | --- | --- |
| mode | switch |  | Mode : switch ou presence |
| icon | fas fa-plug |  | Icône FontAwesome |
| title |  |  | Nom affiché |
| subtitle | Widget |  | Sous-titre |
| accent | #62d5ff |  | Couleur accent |


## 🏠 Utilisation dans MyHome

1. Ouvrir le projet MyHome.
2. Aller dans l'onglet **Palette**.
3. Choisir la page cible.
4. Ajouter le widget **custom_horizontal_widget**.
5. Ouvrir le sous-plan Jeedom.
6. Positionner et redimensionner le widget.
