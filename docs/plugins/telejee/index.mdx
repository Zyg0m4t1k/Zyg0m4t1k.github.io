Le plugin **Telejee** permet de centraliser dans Jeedom :

- un **guide des programmes TV** basé sur des données **XMLTV**
- la création de **chaînes TV comme équipements Jeedom**
- des **actions automatiques** liées à une émission ou à un clic sur un logo
- des **télécommandes virtuelles** avec plusieurs layouts

L’objectif est simple : regrouper l’**EPG**, l’**accès aux chaînes** et les **commandes multimédia** dans une seule interface.

---

## ✨ Fonctionnement général

Telejee s’appuie sur un fonctionnement en trois blocs.

### 1. Un index de chaînes

Le plugin maintient un index des chaînes disponibles à partir des données XMLTV.

Cet index sert à :

- afficher les chaînes disponibles dans l’interface de création
- proposer un filtrage par **genre**
- créer rapidement des équipements Jeedom pour les chaînes souhaitées

### 2. Un XMLTV local filtré

Telejee peut synchroniser un **XMLTV filtré selon les chaînes actives** dans Jeedom.

Cela permet de :

- ne conserver que les chaînes réellement utilisées
- accélérer l’exploitation des programmes
- simplifier les mises à jour du guide TV

### 3. Des équipements Jeedom orientés usage

Deux types principaux sont utilisés dans le plugin :

- **Chaînes TV** : une chaîne devient un équipement Jeedom avec son logo et ses informations de programme
- **Télécommandes** : un équipement de type télécommande permet d’associer des actions Jeedom à des boutons virtuels

---

## 🔧 Prérequis

- **Jeedom 4.4 minimum**
- une connexion Internet pour récupérer les données XMLTV
- des commandes Jeedom, virtuelles ou des scénarios à associer aux boutons et aux logos

Compatibilité annoncée :

- Jeedom Smart, Luna, Atlas
- Raspberry Pi
- Docker
- DIY x86 / x64
- Mobile

## ⚙️ Configuration globale du plugin

Une fois le plugin installé, ouvrez :

**Plugins → Multimédia → Telejee → Configuration**

Les options principales sont les suivantes.

| Paramètre | Description |
|---|---|
| **Forcer la mise à jour des données** | Lance une mise à jour complète des données XMLTV |
| **Sync XMLTV filtré selon chaînes actives** | Regénère un XMLTV local limité aux chaînes actives dans Jeedom |
| **Reconstruire l’index des chaînes disponibles** | Reconstruit le fichier d’index des chaînes si la liste semble incomplète ou incohérente |
| **Activer l’affichage par ordre** | Permet de gérer un affichage ordonné des chaînes |
| **Code localisation** | Permet d’adapter le fuseau horaire si besoin |
| **Activer la gestion des utilisateurs** | Permet de limiter les chaînes visibles selon l’utilisateur Jeedom |

## 📺 Création et gestion des chaînes

### Ajouter des chaînes

Depuis la page principale du plugin, cliquez sur **Chaînes**.

Une fenêtre permet alors de :

- choisir un **genre** ou afficher toutes les chaînes
- charger la liste disponible
- filtrer la liste avec une **recherche**
- cocher plusieurs chaînes
- créer automatiquement les équipements Jeedom correspondants

Chaque chaîne créée utilise comme base son **identifiant XMLTV** et récupère son **logo**.

<!-- screenshot: fenêtre de création des chaînes avec filtre genre, recherche, liste et cases à cocher -->
<!-- path: /img/plugins/telejee/channels_create.png -->
<!-- ![Création des chaînes dans Telejee](/img/plugins/telejee/channels_create.png) -->

### Ce qui est créé pour une chaîne

Lorsqu’une chaîne est ajoutée, Telejee crée un équipement de type chaîne avec :

- son **nom**
- son **logo**
- son identifiant XMLTV
- des commandes d’information programme :
  - **En ce moment**
  - **À venir**
  - **Ce soir**
  - **En fin de soirée**

Ces informations sont ensuite mises à jour automatiquement à partir du XMLTV local.

### Action sur le logo de la chaîne

Chaque chaîne peut exécuter une ou plusieurs actions au clic sur son logo.

Exemples d’usage :

- zapper sur une chaîne
- lancer un scénario multimédia
- envoyer une commande vers une box, une TV ou un ampli

Vous pouvez aussi définir plusieurs actions avec options Jeedom.

## 🗓️ Guide TV et programmes

Telejee exploite les données XMLTV pour alimenter les chaînes et le guide TV.

### Informations disponibles

Selon les données présentes, un programme peut contenir :

- le **titre**
- l’**heure**
- la **durée**
- le **genre**
- le **résumé**
- une **image / icône**
- un **lien**

### Vue programme

Le plugin maintient en pratique deux grands types d’informations :

- **maintenant / à venir**
- **soirée / fin de soirée**

Cela permet un affichage rapide sur les équipements chaînes et dans les interfaces dédiées.

## 🔎 Recherche d’émission et automatisations

Chaque équipement chaîne peut aussi être configuré pour **surveiller un ou plusieurs titres** d’émission.

Dans la configuration de la chaîne, vous pouvez renseigner :

- un ou plusieurs textes de **recherche** séparés par des virgules
- une **plage horaire** pendant laquelle les actions sont autorisées
- des actions à exécuter :
  - **au début** de l’émission
  - **à la fin** de l’émission
- un délai optionnel en **minutes** pour décaler ces actions

Exemples d’usage :

- lancer automatiquement une ambiance quand une émission commence
- envoyer une notification au début d’un film
- exécuter une action quand un programme se termine

Quatre layouts sont disponibles :

- **Universelle**
- **TV**
- **Design**

Pour chaque télécommande, vous pouvez associer des actions Jeedom à de nombreux boutons.

## ⚡ Actions disponibles sur les télécommandes

Selon le layout, vous pouvez rattacher des actions Jeedom à différents groupes de boutons, par exemple :

- **Power ON / OFF**
- navigation
- lecture / pause / stop
- volume
- changement de chaîne
- boutons contextuels
- boutons spécifiques au mode paysage

Le plugin masque les commandes techniques dans l’interface Jeedom et les utilise comme base pour piloter la télécommande visuelle.

---

## 👤 Gestion par utilisateur

Si l’option **Utilisateurs** est activée dans la configuration globale, Telejee peut adapter la liste des chaînes visibles selon l’utilisateur connecté.

Les profils **admin** conservent l’accès complet.

Cela peut être utile pour :

- simplifier l’interface pour certains utilisateurs
- masquer certaines chaînes
- personnaliser l’affichage selon les profils Jeedom

## 🧩 Conseils d’usage

### Pour un usage simple

- créez uniquement les chaînes utiles
- associez une action au clic sur le logo
- utilisez le XMLTV filtré pour garder une base légère

### Pour un usage plus avancé

- configurez des recherches d’émissions
- ajoutez des actions au début et à la fin des programmes
- créez plusieurs télécommandes selon les pièces ou les usages

### En cas de liste incohérente

Ordre conseillé :

1. **Reconstruire l’index des chaînes**
2. **Synchroniser le XMLTV filtré**
3. **Forcer la mise à jour des données** si nécessaire

## 📚 Résumé

Telejee est un plugin multimédia orienté **TV et pilotage** qui combine :

- la récupération et l’exploitation des **programmes TV**
- la création de **chaînes pilotables** dans Jeedom
- des **automatisations** liées aux émissions
- des **télécommandes virtuelles** avec plusieurs présentations

C’est une approche pratique pour réunir dans Jeedom à la fois le **guide TV**, les **actions sur les chaînes** et le **contrôle multimédia**.
