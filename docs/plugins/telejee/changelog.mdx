# Changelog

## v1.0.0 — 2026-03-16

- 🎉 Version initiale du plugin
- Support Samsung Tizen, LG webOS, Sony Bravia, Android TV
- Commandes : alimentation, volume, navigation, lecture, applications
- Widget dashboard et mobile
- Appairage sécurisé par code PIN
- Wake-on-LAN pour l'allumage depuis l'état éteint
- Polling configurable pour la mise à jour des états
