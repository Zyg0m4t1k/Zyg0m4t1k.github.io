## 2026-04-29

- Ajout du type d'équipement **Trajet** : calcul de durée et distance routière via l'API Mapbox Directions
- Ajout de la clé API Mapbox dans la configuration globale du plugin
- Nouveau widget dédié aux trajets (durée, distance, heure de calcul, icône selon profil)
- Profils de trajet disponibles : voiture avec trafic, voiture, vélo, piéton
- Support départ/arrivée dynamique via commande GPS Jeedom ou coordonnées fixes
- Géocodage d'adresse intégré dans l'interface de configuration des trajets
- Commandes créées automatiquement : Durée, Distance, Statut, Calculer

## 2026-04-26

- Réintégration de **MapQuest**  / Nominatim avec gestion des contraintes (clé API + limites)
- Correction d'une erreur JavaScript impactant l'interface
- Mise à jour des icônes pour une meilleure cohérence visuelle
- Amélioration globale de la stabilité du reverse geocoding

## 2026-03-30

- Remplacement de **MapQuest** par **Nominatim (OpenStreetMap)**, suppression de la clé API
- Reverse geocoding désormais **gratuit**
- Ajout d'un **User-Agent unique** par installation (install_id + version + host)
- Mise en place d'une **file d'attente interne** avec traitement via cron
- Ajout d'un **délai entre les requêtes** pour respecter les limites Nominatim
- Optimisation du déclenchement du reverse (uniquement si adresse absente ou incomplète)
- Nettoyage de la configuration et suppression des références MapQuest
