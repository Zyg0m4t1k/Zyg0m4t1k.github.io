## 2026-06-01 ( Beta )

- Nouveau formulaire 2FA intégré : lors de l'authentification, le code est saisi directement dans l'interface avec un compte à rebours de 90 secondes
- le rafraîchissement des appareils réutilise les cookies iCloud existants sans re-authentification, rendant les mises à jour beaucoup plus
  rapides
- Patches automatiques icloudpy à l'installation : support des app-specific passwords, gestion des endpoints de partition Apple
- Virtualenv recréé entièrement à chaque installation pour éviter les problèmes de mise à jour de dépendances
- Timeout d'authentification porté à 10 minutes
- Bouton "Renouveler la session" : demande de confirmation et relance automatiquement l'authentification
- Le dossier data/icloudpy_session/ est maintenant créé automatiquement à la première utilisation
