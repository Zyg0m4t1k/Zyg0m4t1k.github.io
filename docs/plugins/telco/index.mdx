Le plugin **Telecommandes** permet de créer des **télécommandes virtuelles personnalisables** dans Jeedom.

Chaque bouton peut déclencher :

- une **commande**
- une **action Jeedom**
- un **scénario**
- une **automation**

Cela permet de regrouper plusieurs actions dans une **interface simple et rapide à utiliser**, idéale pour un **dashboard**, un **design**, ou un **écran tactile**.

---

## ✨ Vue d’ensemble

Telco permet de construire facilement des interfaces de contrôle.

Exemples d’utilisation :

- piloter une **TV ou box multimédia**
- créer une **télécommande pour un système audio**
- centraliser des **actions domotiques fréquentes**
- créer un **panneau de contrôle personnalisé**

Chaque bouton agit comme un **déclencheur d’action Jeedom**.

---

## ⚙️ Installation

L’installation se fait comme pour tout plugin Jeedom.

### Installation depuis le Market

1. Installez le plugin depuis le Market Jeedom  
2. Activez le plugin
3. Plugins → Organisation → Telco



---

## 🧩 Création d’une télécommande

1. Cliquez sur **Ajouter**
2. Donnez un **nom à votre télécommande**
3. Choisissez un **objet parent**
4. Sélectionnez un **type de télécommande**

![Création équipement](/img/plugins/telco/create.png)

---

## 🧩 Types de télécommandes

Le plugin propose deux modes de fonctionnement.

---

### Télécommande multimédia

Ce mode peut génèrer automatiquement une télécommande classique.


Chaque bouton peut être personnalisée dans la configuration de l'équipement 

![Télécommande multimédia](/img/plugins/telco/multimedia.png)

---

### Télécommande personnalisée

Le mode **Custom** permet de créer vos propres boutons.

Chaque bouton peut avoir :

- un **nom**
- une **icône**
- une **couleur**
- une **action à exécuter**
- une **visibilité configurable**

Cela permet de créer des **interfaces totalement personnalisées**.

![Télécommande personnalisée](/img/plugins/telco/telcoperso.png)


---

## 🧠 Fonctionnement

Lorsqu’un bouton est pressé, le plugin exécute l’action configurée dans l'équipement.


Cela permet de lancer :

- commandes
- scénarios


---

## 🖥 Interface

Le plugin fournit une interface simple :

- gestion des télécommandes
- configuration des boutons
- association des actions

Chaque télécommande peut être affichée dans :

- le **dashboard**
- un **design**
- la **vue mobile **
