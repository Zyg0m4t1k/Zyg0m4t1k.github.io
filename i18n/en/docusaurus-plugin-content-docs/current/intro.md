# Welcome to the documentation of my plugins

A clear and efficient hub for **my Jeedom plugins**: installation, configuration, best practices, troubleshooting, and compatibility (Jeedom 4.4+ / Debian Trixie).

---

## What you’ll find here (when needed)

✅ **Installation guides** (prerequisites, dependencies, venv, permissions)
✅ **Step-by-step configuration** with screenshots / examples
✅ **Commands & scenarios** (values, triggers, usage)
✅ **Troubleshooting** (logs, common errors, health checks)
✅ **Compatibility** (minimum Core version, OS, PHP, migrations)

---

## Get started in 2 minutes

### 1) Choose your plugin
Go to the **Plugins** menu on the left, then open the plugin page.

### 2) If something goes wrong: check logs first
In Jeedom, collect:
- **Plugin logs**
- **http.error** for UI/AJAX issues
- **daemon logs** for Python/venv issues

---

## Contribute / Report a bug

If you want to report an issue, please provide:
- Jeedom version + OS
- Plugin version (stable/beta branch)
- Log extract (without sensitive data)
- Steps to reproduce
