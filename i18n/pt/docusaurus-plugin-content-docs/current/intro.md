# Bienvenue sur la Doc dee mes plugins

Un hub clair et rapide pour **mes plugins Jeedom** : installation, configuration, bonnes pratiques, dépannage et compatibilité (Jeedom 4.4+ / Debian Trixie).

---

## Ce que tu trouveras ici (si nécessaire)

✅ **Guides d’installation** (pré-requis, dépendances, venv, permissions)
✅ **Configuration pas à pas** + captures / exemples
✅ **Commandes & scénarios** (valeurs, déclencheurs, usages)
✅ **Dépannage** (logs, erreurs fréquentes, checks santé)
✅ **Compatibilité** (Core min, OS min, PHP min, migrations)

---

## Démarrer en 2 minutes

### 1) Choisis ton plugin
Va dans le menu **Plugins** à gauche, puis ouvre la fiche du plugin.

### 2) En cas de souci : logs d’abord
Dans Jeedom, récupère :
- **Logs du plugin**
- **http.error** si UI/AJAX
- **daemon** si Python/venv


---

## Contribuer / Remonter un bug

Si tu veux remonter un souci, donne :
- Version Jeedom + OS
- Version du plugin (branche stable/beta)
- Extrait log (sans données sensibles)
- Étapes pour reproduire
